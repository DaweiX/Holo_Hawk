# Holo_Hawk "智信鹰眼"   
## 摘要：  
一个基于混合现实（HoloLens），人工智能（Yolo & Tegra X2）与区块链（RPi轻量级实现）的自组网系统。
  
## 主要功能：  
* 基于区块链思想的可信定位
* 重要目标识别、标记

## 主要特色：
* 离线识别
* 数据可信
* 双重定位

## 文件说明：
* Blockchain文件夹包含在树莓派上的区块链实现。
* HoloLens文件夹包含HoloLens的功能代码。（需要使用Unity新建项目导入代码并输出UWP工程）
* Recognize包含在Nvidia Tegra X2上的识别基本功能实现。（Yolo，Facenet等代码请自行下载）
